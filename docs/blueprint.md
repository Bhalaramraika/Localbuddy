# **App Name**: Local Buddy Pro

## Core Features:

- Google Login: Allow users to log in using their Google account.
- Profile Creation and Verification: Enable users to create profiles with name, photo, and location, and verify their mobile number (OTP) and Aadhar/ID for enhanced security.
- Task Posting Wizard: Implement a step-by-step wizard for posting tasks, including category selection, title, location, deadline, and budget, with a suggested price feature.
- Escrow System: Manage payments using an escrow system that locks funds upon task posting and releases them to the Buddy upon completion, ensuring secure transactions.
- Live Chat with Safety Features: Facilitate real-time chat between Poster and Buddy with safety warnings, masked call buttons, SOS emergency button, voice note support, image uploads, and smart replies. The SOS button will share the live location with a trusted contact.
- Level and Badge System: Implement a level system (Rookie, Pro, Local Hero) with XP points for completed tasks and achievement badges to gamify the experience and encourage user engagement.
- Task Marketplace with Search and Filters: Create a home screen displaying tasks with user details, titles, prices, details, and action buttons. Include search functionality and category filters, and a map toggle between list and map views.

## Style Guidelines:

- Background color: Transparent blue (#ADD8E6) to provide a glassy effect.
- Primary color: Trust Green (#10B981) for 'Accept', 'Post Task', and 'Release Payment' buttons.
- Accent color: Active Blue (#3B82F6) for links, active chat, and info elements.
- Secondary accent color: Gold/Yellow (#F59E0B) to highlight 'Level Up,' 'Badges,' and 'Escrow Locked Money'.
- Alert color: Alert Red (#EF4444) for the 'SOS Button,' 'Delete' actions, and 'Report Dispute' options.
- Body and headline font: 'Inter', a grotesque-style sans-serif font.
- Subtle animations like fade, jump, breathing, and blur effects throughout the app for a dynamic user experience.
- Glassmorphism: Apply a frosted glass effect to cards and buttons for a modern and premium look.
- Mobile UX: Ensure all buttons are thumb-friendly with a bottom navigation bar for Home, Wallet, Chat, and Profile.
- Auto Dark Mode: Implement automatic switching between Light Mode during the day and Dark Mode at night, based on the phone's settings.
- Notification: No popup alert, use toast notification
- Clear and intuitive icons for all major actions, such as wallet transactions, chat functions, and profile settings.
